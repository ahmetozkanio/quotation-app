# Quotation App

A lightweight, browser-based quotation app that stores data in LocalStorage. Minimal setup — open `index.html` or serve the folder.

Basic features
- Company info (logo, address)
- Product library (add/edit/delete)
- Create & save quotations
- Generate PDF from saved quotations
- Export/import data (JSON)

Notes
- Works fully offline in modern browsers
- Data is stored locally (no server)
